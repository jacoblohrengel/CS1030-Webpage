HTML5 Semantic Elements

Elements like <header>, <nav>, <main>, <section>, and <footer> describe structure and meaning. This helps accessibility tools and makes your code easier to understand.

What do <div> and <span> do?

<div> is a block-level container—great for grouping larger chunks of content for layout or styling.

<span> is an inline container—perfect for styling just a few words inside a sentence without breaking the line.
